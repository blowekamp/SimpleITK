# SimpleITK Release v3.0.0a3

This is an automatic packaging of SimpleITK based on the v3.0.0a3 tag.


To upgrade to this Python binary package run:
```pip install --upgrade --pre simpleitk --find-links https://github.com/SimpleITK/SimpleITK/releases/tag/v3.0.0a3```



**What's Changed**
* Install required locale for  ITK testing by @blowekamp in https://github.com/SimpleITK/SimpleITK/pull/2525
* Add UseLegacyOperatorCoefficients to SobelEdgeDetectionImageFilter by @blowekamp in https://github.com/SimpleITK/SimpleITK/pull/2515
* ENH: Use CMake TIMESTAMP by @blowekamp in https://github.com/SimpleITK/SimpleITK/pull/2527
